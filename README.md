# Expense-tracker_upskill
A console-based Expense Tracker application built using Core Java that helps users manage personal expenses, categories, filtering, and financial reports with data persistence.


##  Overview
A Java console application to track personal expenses efficiently with category-based management.

---

##  Features
- Add Expenses
- Manage Categories
- Filter Expenses (Date/Category/Amount)
- Modify & Delete Expenses
- Monthly Reports
- Data Persistence (File Handling)

---

##  Technologies Used
- Java (Core)
- File Handling
- ArrayList
- OOP Concepts

---

##  Run
```bash
javac ExpenseTracker.java
java ExpenseTracker
